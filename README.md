# Atmega328P Video Transcoder

Aim of this project is to use Atmega238P microprocessor as a video transcoder unit.

Atmega328P characteristics:

| Flash | SRAM | CPU Hz |
|:------|:-----|--------|
| 32 KB | 2 KB | 16 MHz |

And with this beast I will try to decode and encode videos!
